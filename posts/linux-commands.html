权限管理

命令作用示例
chmod修改权限chmod 755 script.sh
chown修改所有者chown user:group file
sudo以管理员执行sudo apt update

权限数字： r=4, w=2, x=1

  • 755 = rwxr-xr-x (文件可执行，目录可浏览)
  • 644 = rw-r--r-- (普通文件)
  • 777 = rwxrwxrwx (所有人可读写执行)

常用组合命令

# 查找并删除大文件
find . -size +100M -delete

# 查找包含特定内容的文件
grep -r "TODO" . --include="*.py"

# 统计代码行数
find . -name "*.py" | xargs wc -l

# 批量重命名
for f in *.jpg; do mv "$f" "prefix_$f"; done

快捷键

快捷键作用
Ctrl + C中断当前命令
Ctrl + Z挂起当前进程
Ctrl + L清屏
Ctrl + A/E光标移到行首/行尾
Ctrl + U/K删除光标前/后内容
!!执行上一条命令
!n执行历史第n条命令
记住这些命令最有效的方法是实际使用。创建一个 .bash_aliases 文件，把常用但长的命令设置别名会更方便。
